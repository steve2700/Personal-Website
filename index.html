Portfolio

ALX AirBnB Clone

The ALX hbnb project is a Airbnb-like web application that we built at ALX School as we progressed through the curriculum. For the first part of Hbnb, we created custom classes, a file storage engine, and a console for creating, reading, updating, and deleting objects. Object data was serialized and saved into a JSON file and the data was deserialized from the file each time the console was run. We also worked on responsive static web pages. For the second part of Hbnb, we added database storage to the storage engine and mapped the custom classes to tables in the database. For the third part of Hbnb, we added a custom RESTful API for retrieving data from the storage engine. The final part of Hbnb was creating dynamic web pages where a user could filter data from the storage engine.
